09 / Motion & Animation

Subtle.
Refined.

Motion in the Rev Whisper system is never decorative. It serves a purpose: guiding attention, confirming actions, and making interfaces feel responsive without being distracting.

Principles

Three rules for movement.

Every animation in the system follows these principles. If a motion doesn't serve one of these purposes, remove it.

01
Purposeful
Every animation guides the user's attention or confirms an action. Nothing moves for decoration. If it doesn't help the user, it doesn't move.
02
Subtle
Animations are felt more than seen. Gentle fades, soft position shifts, quiet scaling. The user should never be waiting for an animation to finish.
03
Consistent
The same easing curve and duration range across the entire product. Consistency in motion builds subconscious trust. No surprises.
Specifications

Timing and easing.

Duration
Micro-interactions (button states, toggles) use 150ms. Element transitions (cards, panels, modals) use 250ms. Page-level transitions use 400ms. Never exceed 500ms.
150ms / 250ms / 400ms
Easing
Use ease-out for elements entering the viewport. Use ease-in for elements leaving. Use ease-in-out for elements that transform in place. Never use linear easing.
cubic-bezier(0.25, 0.1, 0.25, 1.0)
Opacity
Fade-in is the default entrance animation. Elements fade from 0 to 1 with a subtle 8px upward translate. Clean, predictable, and unobtrusive.
opacity: 0 → 1 + translateY(8px → 0)
Hover States
Interactive elements respond with a 150ms background color transition. Buttons slightly darken. Cards get a subtle border color shift. No scaling, no bouncing.
transition: all 150ms ease
Motion Guidelines

What moves and what doesn't.

Do
Use fade + translate for content entering the viewport
Animate background-color on hover for interactive elements
Use consistent durations across the entire product
Respect prefers-reduced-motion for accessibility
Test animations at 2x speed to ensure they feel right at 1x
Don't
Don't use bounce, elastic, or spring easing curves
Don't animate layout properties (width, height, top, left)
Don't create animations that loop or repeat indefinitely
Don't use motion to draw attention to promotional content
Don't exceed 500ms for any single animation

12 / Text Emphasis & Annotation Marks

Mark what
matters.

Hand-drawn emphasis marks that highlight a single idea with warmth and intention. Controlled enough to feel premium. Human enough to feel real.

Design Intent

Why the brand uses emphasis marks.

Rev Whisper is built on data and precision. But precision alone doesn't build trust. It earns respect. What builds trust is the feeling that a real person is behind the algorithm, pointing at the number that matters, circling the insight worth remembering.

Emphasis marks introduce a controlled moment of human texture into an otherwise minimal system. They signal: we know what's important, and we want to make sure you see it too.

These marks are not decoration. They are editorial tools, the visual equivalent of a revenue manager leaning across the table and saying, "This is the number." They add warmth without sacrificing authority. They create focus without introducing noise.

Used correctly, they make the brand feel more considered, not less precise.

Approved Marks

Three marks. Nothing else.

The emphasis system consists of exactly three mark types. Each has a distinct purpose. No additional marks may be introduced without brand approval.

12.3%
Circle
Draws the eye to a single data point, metric, or key phrase. The primary emphasis mark. Use it most.
revenue
Underline
A hand-drawn underline beneath a single word or short phrase. Subtler than the circle. Use for secondary emphasis.
optimized
Star
A small, hand-drawn star placed at the top-right of a word. Reserved for moments of achievement or milestone emphasis. Use sparingly.
Usage Rules

When to mark. When to leave it.

Emphasis marks exist to direct attention, not to decorate. If the content hierarchy already does the work, the mark is unnecessary. When in doubt, leave it out.

Do
Use marks when a single data point or metric is the most important element on the page
Draw the eye to a key phrase within a dense content block
Mark revenue milestones or performance achievements that deserve visual weight
Highlight the word that carries the entire meaning of a headline
Reinforce the typographic hierarchy rather than competing with it
Don't
Don't add marks when the content is already clear without additional visual cues
Don't use more than one mark in the same view or component
Don't apply marks to body copy, captions, or small text below 16px
Don't use marks for decoration. Every mark must serve an editorial purpose
Don't mark content in layouts that are already visually active or information-dense
Consistency Rules

Built on the same precision.

Emphasis marks follow the same design language as the brand icon and UI iconography. Stroke weight, color, and spacing are fixed. Never improvise.

1.8px
Stroke Weight
All emphasis marks use a 1.8px stroke weight. This matches the brand icon's line quality and feels intentional at every scale. Never go thinner or thicker.
Fern
Color
Emphasis marks are always rendered in Fern (#4A6741). This maintains the brand rule: green is reserved for moments of strategic emphasis. No other colors.
8px
Clearance
Maintain a minimum 8px gap between the mark and the text it emphasizes. The mark should feel adjacent, not touching. Give the text room to breathe.
Round
Line Caps
Round line caps on all strokes. This matches the speech-wave icon and the UI icon system. It keeps the marks feeling warm and organic, not mechanical.
1–3°
Rotation
Circles and underlines may carry a subtle 1–3 degree rotation to convey a hand-drawn quality. Never exceed 3 degrees. The imperfection should feel deliberate.
1x
Scale Ratio
Circles scale proportionally to the text they enclose. Underlines match the width of the word. Stars are fixed at 12–16px. Never scale marks independently.
Hierarchy

How marks interact with type.

Emphasis marks are subordinate to the typographic hierarchy. They never replace size, weight, or color as the primary means of creating visual structure. They accent. They do not lead.

Display / H1
Maximize every booking.
At display sizes, marks can appear on a single emphasized word. The mark supports the existing Fern color emphasis already present in headings. Use one technique per heading, not both.
H2 / Subhead
Revenue grew by 18%.
At subhead sizes, circles work well around data points or percentages. The circle should enclose the metric naturally without crowding adjacent words.
Data / Metrics
$12,830
On standalone metrics and revenue figures, the circle is the primary mark. The star may be paired for milestone moments only. Never use both circle and underline on the same element.
Body Copy

Emphasis marks are not used on body copy. At 20px and below, they introduce visual noise that competes with readability. Let the text do the work.

Body copy, captions, labels, and UI text are off-limits for emphasis marks. The typographic system already handles hierarchy at these sizes. Adding marks here would undermine the restraint the brand depends on.
Restraint Rules

The power is in scarcity.

Emphasis marks follow the same logic as the Fern accent color: their impact comes from how rarely they appear. If every element is marked, nothing is emphasized. Treat each mark as a budget you cannot afford to overspend.

1
One mark per view
A single screen, page section, or card should contain no more than one emphasis mark. If two elements compete for emphasis, choose the one that matters more. There is always one.
2
One word or metric
Each mark highlights one word, one number, or one short phrase (three words maximum). Never circle an entire sentence. Never underline a paragraph. The mark is a scalpel, not a highlighter.
3
Earn the mark
Before adding a mark, ask: does this element already have sufficient visual weight through size, color, or position? If the answer is yes, the mark is redundant. Remove it.

Rules for protecting the system.

Do
Use emphasis marks exclusively in Fern (#4A6741) on light backgrounds
Apply marks only to headings, subheadings, or standalone metrics
Maintain the 1.8px stroke weight across all mark types
Limit each page section or card to a single emphasis mark
Treat the marks as editorial signals: purposeful, specific, and rare
Don't
Don't apply marks to body copy, labels, captions, or UI elements
Don't use multiple mark types on the same word or phrase
Don't use marks in colors other than Fern. No Dune, no White, no gradients
Don't animate emphasis marks. They are static and intentional
Don't introduce new mark types. The system is closed: circle, underline, star
Rev Whisper, A Kasu Capital Company Brand Guidelines
